Plot ETOPO1 data with Xarray and Cartopy using Jupyter Notebook

Posted 23 Nov 2020

In this post we will learn how to plot topography and bathymetry data using Cartopy. Although still widely used by many programmers, basemap (the most famous library to plot 2D data on maps) is deprecated since 2017 and with the discontinuation of Python 2.7 itself, Cartopy is becoming the main tool for base mapping with matplotlib! But you should be aware of a few shortcomings when working with (e.g. canâ€™t use latitude/longitude labels for many map projections). If you run in to these issues, basemap is likely to accomplish what you need.

ETOPO1 is an 1 arc-minute global relief model of Earth's surface that integrates land topography and ocean bathymetry. Its data can be easily downloaded from the NCAR's Research Data Archive.

Alternatively, I uploaded into a Github repository the ETOPO1 data (~1 mb) that I used in this script.

In order to make more easier to learn, this script was made using Jupyter Notebook. Also, this I've writen about how to highlight a specific topography (300 and 500 meters) and bathymetry (200 and 1000 meters) lines and to plot a marker and add a text to display a location in the map.

Lastly, a few libraries are required to run this script. Click on each one of the links below to get the code to install them:

  • Numpy
  • Scipy
  • Xarray
  • Cmocean
  • Cartopy
  • Matplotlib
