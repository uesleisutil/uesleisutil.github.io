COAWST Manual
Numerical Modeling

Guia prático para utilização do COAWST - 2º edição

This guide is designed to assist new users with the Coupled Regional Modeling System (COAWST). The main idea is to teach the reader the necessary steps to use COAWST, from its installation, simulation of a test case and the configuration of a project.

To achieve this goal, we use several programming languages, such as Fortran, Python, and MATLAB. In the future we intend to adapt all scripts to Python, Fortran and C++. When we started writing this guide, we wanted to pass on to t he reader our experience in using the COAWST throughout the reading, to understanding its operation and application. However, a major difficulty in this process was the generation of the boundary and initial conditions of the regional ocean model, the Regional Ocean Modeling System (ROMS), which depends on paid software.

To get around this we chose to work with the model2roms package. This set of routines was developed in Python and Fortran language by Trond Kristiansen (http://www.trondkristiansen.com). We also highlight that in some chapters the reader will find out how to use COAWST in a cluster with parallel architecture that is available for use by the National Institute for Space Research (INPE). In this case, the guide may only serve as inspiration and will draw the reader's prior knowledge of applying COAWST to their own computer system. In this second edition, we highlight the update to COAWST v.3.4, the introduction of a chapter on the sea ice model, and the renewal of the model2roms toolkit to generate model input data and the structural reorganization Guide.

model2roms

Model2roms is a Python-based toolbox for creating the necessary grid, climatology, boundary, and initial forcing files required to run the ROMS (Regional Ocean Modeling System) model. This project is forked from the original work of Trond Kristiansen, which can be accessed here.
